## 🎯 Descripción

**ALEN.IA** es una aplicación de escritorio profesional para la gestión integral de inventario y ventas, diseñada específicamente para pequeños y medianos comercios. Desarrollada con tecnología Python y una interfaz moderna, ofrece todas las herramientas necesarias para optimizar la gestión comercial de manera eficiente y sin costo.

### 💡 ¿Por qué elegir ALEN.IA?

- **100% GRATUITO** - Sin licencias, sin mensualidades, sin límites
- **INSTALACIÓN INSTANTÁNEA** - Solo descargar y ejecutar
- **SIN INTERNET REQUERIDO** - Funciona completamente offline
- **DATOS SEGUROS** - Tu información siempre bajo tu control
- **INTERFAZ MODERNA** - Diseño profesional y fácil de usar

## 🚀 Inicio Rápido

### 🎯 **Primera Configuración (5 minutos)**

#### **Paso 1: Configurar Inventario**
1. Abre la aplicación
2. Ve a **"⚙️ Menú Gestión"**
3. Selecciona **"📂 Carga Masiva de Productos"**
4. Descarga el archivo modelo CSV
5. Completa con tus productos
6. Sube el archivo

#### **Paso 2: Primera Venta**
1. Regresa al menú principal
2. Haz clic en **"💰 Nueva Venta"**
3. Selecciona un producto del inventario
4. Agrega cantidad y forma de pago
5. Haz clic en **"Finalizar Venta"**

#### **Paso 3: Revisar Resultados**
1. Ve a **"📊 Ventas del Día"**
2. Verifica que la venta se registró correctamente
3. Al final del día, usa **"🏦 Cierre de Caja"**

### 📋 **Flujo de Trabajo Diario**

## 🆘 Soporte y Ayuda

### ❓ **Preguntas Frecuentes**

#### **❌ La aplicación no abre**
**Solución:** 
1. Clic derecho en el ejecutable → "Ejecutar como administrador"
2. Verificar que Windows no esté bloqueando el archivo
3. Desactivar temporalmente el antivirus

#### **❌ No se ven los logos**
**Solución:** 
1. Verificar que los archivos `.png` estén en la misma carpeta
2. No mover el ejecutable sin los archivos de recursos
3. Reinstalar desde el ZIP original

#### **❌ Los datos no se guardan**
**Solución:** 
1. Ejecutar como administrador
2. Verificar permisos de escritura en la carpeta
3. Cambiar a una ubicación con permisos completos

#### **❌ Error al cargar productos**
**Solución:** 
1. Verificar formato del archivo CSV
2. Eliminar `productos.json` para recrearlo
3. Usar la plantilla modelo incluida
